Details

Each year, four main Standard-legal sets are released by Wizards of the Coast.

Standard-legal sets are added to Standard on release, and are removed two "rotations" later. Rotations are annual, typically coinciding with the release of the fall set in September or October. This means most sets remain in Standard for 15-24 months.

A Standard deck contains at least 60 Standard cards, plus an optional sideboard of up to 15 Standard cards. No more than 4 copies of a card may be included, except for basic land cards: Plains, Islands, Swamps, Mountains, and Forests.

Full rules are available on the official Magic: the Gathering website here.

Any copy of a Standard card can be included in a Standard deck, even if that copy is from a non-Standard set. For instance, a Standard deck may include a 1998 copy of Duress from Urza's Saga, because Duress was also included in Core Set 2020.

Getting Cards for Standard

Most Standard cards can be opened in booster packs of the Standard-legal sets listed above. Many game stores also carry a selection of Standard cards that can be purchased individually.

Aside from booster packs, there are several other products that contain a selection of only Standard cards:

  • Preconstructed "Planeswalker Decks" for Standard-legal sets
  • Preconstructed 2019 and 2020 "Welcome Decks" given away by game stores
  • Preconstructed 2019 "Challenger Decks"

Related Formats

Brawl is a format which uses the same Standard-legal set rotation as Standard. Brawl is often played casually with 3 or more players per game, and uses different rules for deck construction and gameplay.
Read more about Brawl on the offical Magic: the Gathering website here.

Modern is a format which allows cards from any Standard-Legal set released since 2003's 8th Edition, as well as the non-Standard-legal set Modern Horizons. Every card added to Standard is also added to Modern. Sets never rotate out of Modern, but numerous cards are included on the Modern banlist.
Read more about Modern on the official Magic: the Gathering website here.
