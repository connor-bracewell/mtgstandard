Details

Standard consists of cards from recent Standard-legal sets. "Standard-legal" is a designation applied by Wizards of the Coast to the main card sets released each year (typically four per year, distributed evenly).

Every Standard-legal set is added to Standard when it is released, and is removed from Standard two "rotations" later. One rotation occurs each year, typically coinciding with the release of the Fall set in September or October. This means most sets remain in Standard for 15-24 months.

Sometimes, cards included in Standard-legal sets may mave been included in previously-released sets. For example, Duress was printed in M20, M19, and many other sets dating back to 1998. A player may use a any copy of such a card in their Standard deck, regardless of when it was originally printed.

There are additional rules for constructing decks for Standard, such as the minimum number of required cards. See the full rules on the official Magic: the Gathering website here.

Some sets are not Standard-legal, and are not added to Standard on release. Some cards from these sets are also available in Standard sets (such as Giant Growth in both War of the Spark and Battlebond) but any other cards cannot be used in Standard.

Upcoming sets which contain non-Standard cards:

  • Commander 2019 Preconstructed Decks

Some non-"Booster Pack" products contain Standard-legal cards only, and can be used in Standard. Recent examples of such products include:

  • Preconstructed "Planeswalker Decks" from Standard-legal sets
  • Preconstructed 2019 "Challenger Decks"
  • Preconstructed 2019 and 2020 "Welcome Decks" given away by game stores

Note that cards received at pre-release events are still not considered "released" and may not be used in Standard until the date of the set's official release.

Related Formats

Brawl is a format which uses the same Standard-legal set rotation as Standard. Brawl is typically played with 3 or more players per game, and uses different rules for deck construction and gameplay. Read more about Brawl on the offical Magic: the Gathering website here.

Modern is a format where players can use cards from any Standard-Legal set released since 8th Edition (2003-07-28), as well as the Modern-only set Modern Horizons. Sets of cards are never removed from Modern, but numerous cards are included on the Modern banlist. Every current Standard deck is technically a legal (but low-powered) Modern deck, and will continue to be legal in Modern after its cards rotate out of Standard. Read more about Modern on the official Magic: the Gathering website here.
